# Makebread

Bread machine recipe manager with timer calculations.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install makebread
```

### Fedora/RPM
```bash
sudo dnf install makebread
```
